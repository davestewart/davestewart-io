An introduction to Form Template and Form Builder

What it is

Form Builder is a set of standards-compliant HTML, CSS and JavaScript files that work together to create a versatile, attractive and functional form experience.

Issues such as layout, tabs, validation, highlighting and data-structure are taken care of.

One more very important point to mention is that Form Builder has been designed to work PHPs handling of multi-dimensional arrays - that is, using form controls that are named with square brackets, so that the data is automatically parsed into a multi-dimensional array.

How it works

After downloading some template files, you use the Form Builder to create the HTML for each of your controls.

By simply typing in a list of simple labels, for exmaple "First Name, Last Name, Address", you then simply click to create the HTML for the appropriate form controls (eg. text, radio, dropdown), which can then be copied and pasted into the template.

The markup created can be customzed with the many options provided; for example: control types, element types, naming conventions, etc.

Getting started

Browse the controls by clicking the tabs above, then when you're ready download the template files, and go online to the Form Builder to start creating the HTML.

Note: Form template and Form Builder can be considered a work in progress, and will certainly be updated from time to time.

Brief instructions on how to use the tab functionality

Tabs

The tabs at the top of this page are simply a bulleted-list, with some links and basic styling.

Each tab <li>has a "section" attribute, which must correspond to the id of an element in the page, eg.

section="#section-main-template"

When the tab is clicked, and the section corresponding to the section attribute exists, it will be shown.
